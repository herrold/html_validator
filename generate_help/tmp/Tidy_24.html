Tidy: <a> is probably intended as </a>

[edit]

Cause:

The closing tag for a link is missing its slash.

[edit]

Example:

<a href="http://example.com/">An example site<a>
<a href="http://example.com/">An example site</a>
[edit]

Solution:

Add a slash to the closing tag.
