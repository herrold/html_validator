Tidy: missing <...>

[edit]

Cause:

A tag is missing in a mandatory context.

[edit]

Example:

missing <tr>

<table>

 <td>abc</td>
</table>

<table>

 <tr>
  <td>abc</td>
 </tr>
</table>

[edit]

Solution:

Check in which context should the tag be defined.
