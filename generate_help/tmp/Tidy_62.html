Tidy: <...> converting backslash in URI to slash

[edit]

Cause:

An attribute of type URI contains a backslash, it is converted to a slash.

[edit]

Example:

<a href="http://www.domain.com\dir">
<a href="http://www.domain.com/dir">
[edit]

Solution:

Use / in URI in place of \
