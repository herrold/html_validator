Tidy: <...> lacks "..." attribute

[edit]

Cause:

An HTML tag has a mandatory attribute, but this attribute is missing.

[edit]

Example:

<form> lacks "action" attribute

<form>
<form action="my_action.jsp">
[edit]

Solution:

Consult the tag definition. Add the missing attribute.
