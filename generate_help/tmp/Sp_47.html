OpenSP: End of document in prolog

[edit]

Cause:

xml prolog at the beginning of the document was not closed, or the file is empty.

[edit]

Example:

<?xml version="1.0" encoding="UTF-8"?>

<?xml version="1.0" encoding="UTF-8"

[edit]

Solution:

Make sure you have closed the prolog
