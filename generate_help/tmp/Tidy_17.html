Tidy: Insert missing <title> element

[edit]

Cause:

The <title> element is required in all HTML and XHTML documents. This error is caused by a missing <title> element.

[edit]

Solution:

Insert the missing <title> element in the head section.

[edit]

Example: 

<head>
  <title>Mozilla</title>
</head>
