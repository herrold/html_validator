OpenSP: DTDs other than base allowed only if CONCUR YES or EXPLICIT YES

[edit]

Cause:

This eror can happens when 2 DTDs are declared in a page.

[edit]

Example:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd" >
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/loose.dtd">
<html>
....

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd" >
<html>
....

[edit]

Solution:

Remove one of the declaration.
