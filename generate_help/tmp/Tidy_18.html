Tidy: Multiple <frameset> elements

[edit]

Cause:

HTML element <frameset> was used more than one times without interleaving. You can use multiple <frameset>-elements only nested.

[edit]

Example:

<frameset ...><frame ...></frameset>

<frameset ...><frame ...></frameset>

<frameset ...><frame ...>

<frameset ...><frame ...></frameset>
</frameset>

[edit]

Solution:

Use nested <frameset>-elements.
