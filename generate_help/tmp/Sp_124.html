OpenSP: Value of attribute ... invalid: ... cannot start a number token

[edit]

Cause:

The value of this attribute should be a number, and you probably used a wrong syntax.

[edit]

Example:

textexample textexample
textexample textexample
[edit]

Solution:

textexample textexample
